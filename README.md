# Saraban Downloader 📥

**Saraban Downloader** เป็น Chrome Extension สำหรับช่วยดาวน์โหลดเอกสาร PDF อัตโนมัติทีละหลายแถวจากระบบสารบรรณอิเล็กทรอนิกส์ ช่วยลดเวลาและขั้นตอนการทำงานที่ต้องคอยกดเปิดทีละหน้าและบันทึกไฟล์ด้วยตนเอง

---

## ฟีเจอร์เด่น (Key Features)

- **ดาวน์โหลดอัตโนมัติเป็นชุด (Batch Auto-download):** ดึงข้อมูลและดาวน์โหลด PDF อัตโนมัติจากตารางหลักโดยระบุจำนวนแถวที่ต้องการ หรือเลือกดาวน์โหลดทั้งหมดได้ทันที
- **ตั้งชื่อไฟล์อัตโนมัติ (Auto-naming):** ตั้งชื่อไฟล์เอกสาร PDF ที่ดาวน์โหลดตาม "หัวข้อเรื่อง" ในระบบโดยอัตโนมัติ และทำความสะอาดอักขระพิเศษที่ไม่ได้รับอนุญาตในชื่อไฟล์ (`\ / : * ? " < > |`) ให้เป็นขีดล่าง (`_`)
- **การหน่วงเวลาป้องกันระบบตรวจจับ (Smart Delay):** ตั้งค่าเวลาหน่วง (Delay) ระหว่างการดาวน์โหลดแต่ละแถวได้ เพื่อป้องกันไม่ให้ส่งคำขอถี่เกินไปและลดภาระการทำงานของเซิร์ฟเวอร์
- **Activity Log & Progress:** แสดงสถานะการดาวน์โหลด แถบความคืบหน้า (Progress Bar) และบันทึกกิจกรรม (Logs) แบบเรียลไทม์ในระหว่างประมวลผล เพื่อให้ทราบความคืบหน้าของแต่ละแถวอย่างละเอียด
- **ควบคุมการทำงานได้ตลอดเวลา (Stop/Resume Control):** มีปุ่มสั่งหยุดการทำงานได้ทันทีเมื่อต้องการ

## โครงสร้างโปรเจกต์ (Project Structure)

├── content.js         # สคริปต์สแกน ค้นหาข้อมูล และควบคุมหน้าเว็บ (Automation Engine)

## วิธีการติดตั้ง (Installation Guide)

เนื่องจากส่วนขยายนี้เป็นเวอร์ชันพัฒนาภายใน (Developer Mode) สามารถติดตั้งบน Google Chrome ได้ตามขั้นตอนดังนี้:

1. **ดาวน์โหลดโค้ด**ของโปรเจกต์นี้ลงในเครื่องคอมพิวเตอร์ของคุณ
2. เปิดเบราว์เซอร์ **Google Chrome** แล้วไปที่หน้าจัดการส่วนขยายโดยพิมพ์ `chrome://extensions/` ที่ช่องกรอก URL หรือคลิกที่เมนู (จุดสามจุด) -> **เครื่องมือเพิ่มเติม (More Tools)** -> **ส่วนขยาย (Extensions)**
3. เปิดใช้งาน **โหมดนักพัฒนาซอฟต์แวร์ (Developer mode)** ที่มุมขวาบนของหน้าจอ
4. คลิกปุ่ม **โหลดส่วนขยายที่แยกแพ็กเกจแล้ว (Load unpacked)** ที่มุมซ้ายบน
5. เลือกโฟลเดอร์โปรเจกต์ `saraban-downloader` (โฟลเดอร์ที่มีไฟล์ `manifest.json` อยู่ข้างใน)
6. เมื่อติดตั้งเสร็จสิ้น ไอคอนของส่วนขยายจะปรากฏบนแถบเครื่องมือของ Chrome

## วิธีการใช้งาน (How to Use)

1. เข้าสู่ระบบสารบรรณอิเล็กทรอนิกส์และเปิดหน้าตารางเอกสารที่ต้องการดาวน์โหลด
2. คลิกที่ไอคอนของส่วนขยาย **Saraban Downloader** บนแถบเครื่องมือ
3. **ตั้งค่าพารามิเตอร์**ที่ต้องการในแถบการตั้งค่า:
   - **จำนวนแถวที่ดาวน์โหลด:** กรอกจำนวนแถวที่ต้องการดาวน์โหลดจากตาราง (เช่น 10 แถว) หรือเลือกติ๊กช่อง **"ทั้งหมด"** หากต้องการดาวน์โหลดทุกแถวที่แสดงอยู่
   - **Delay ระหว่างแถว:** กำหนดเวลาพักระหว่างดาวน์โหลดแต่ละแถว (หน่วยเป็นมิลลิวินาที แนะนำให้ตั้งไว้ที่ `1500` ms หรือมากกว่าเพื่อความปลอดภัย)
4. คลิกปุ่ม **📥 เริ่มดาวน์โหลด**
5. ระบบจะทำการเลื่อนหน้าจอไปยังแต่ละแถว เปิดหน้าต่างแสดงผล PDF ดึงชื่อหัวข้อเรื่อง ตั้งชื่อไฟล์ และดาวน์โหลดไฟล์ลงในเครื่องของคุณโดยอัตโนมัติทีละรายการ
6. สามารถติดตามสถานะได้จากแถบความคืบหน้าและ **Activity Log** หรือกดปุ่ม **⏹ หยุด** หากต้องการยุติการทำงานก่อนเสร็จสิ้น

## หลักการทำงาน (How It Works)

1. **Popup Interaction:** ผู้ใช้คลิกปุ่ม "เริ่มดาวน์โหลด" จากหน้าต่างควบคุม -> `popup.js` จะส่งคำสั่งไปที่หน้าเว็บผ่าน Message API
2. **Content Script Injection:** ส่วนขยายทำการโหลด `content.js` ลงไปทำงานในสิทธิ์การเข้าถึงหน้าเว็บสารบรรณ
3. **DOM Inspection:** `content.js` จะสแกนตารางเพื่อระบุเป้าหมายในการคลิก และจำลองการคลิกทีละแถวเพื่อเรียกหน้ารายละเอียดเอกสารขึ้นมา
4. **Filename Generation:** ดึงชื่อเรื่องจากแถบรายละเอียด ล้างอักขระพิเศษที่ระบบไฟล์ไม่รองรับ และส่งชื่อไปยังส่วนควบคุมเบื้องหลัง (`background.js`) ผ่าน Message `setNextFilename`
5. **Download interception:** เมื่อหน้าเว็บคลิกเปิดเอกสาร PDF และส่งสัญญาณดาวน์โหลด `background.js` จะดักจับอีเวนต์ดาวน์โหลด (`chrome.downloads.onDeterminingFilename`) และเปลี่ยนชื่อไฟล์ PDF ให้ตรงกับหัวข้อเรื่องที่เตรียมไว้
6. **Progress Tracking:** ส่งข้อมูลอัปเดตสถานะการดาวน์โหลดกลับมาให้ `popup.js` แสดงผลแบบเรียลไทม์จนครบจำนวนแถวที่ระบุ

## ข้อจำกัดและข้อควรระวัง (Limitations & Caveats)

- **การดาวน์โหลดของ Chrome:** โฟลเดอร์ปลายทางของเอกสารทั้งหมดจะถูกบันทึกไว้ในโฟลเดอร์ **Downloads** เริ่มต้นของ Google Chrome ของคุณ
- **การบล็อกการดาวน์โหลดหลายไฟล์ (Multiple Downloads):** ในการใช้งานครั้งแรก Google Chrome อาจแสดงข้อความถามว่าต้องการให้ส่วนขยายนี้ดาวน์โหลดไฟล์หลายไฟล์พร้อมกันหรือไม่ กรุณาตอบ **"อนุญาต" (Allow)** เพื่อให้ระบบดาวน์โหลดได้อย่างต่อเนื่อง
- **โครงสร้างหน้าเว็บ:** ส่วนขยายนี้ขึ้นอยู่กับรูปแบบ HTML ของหน้าเว็บที่ใช้ปฏิบัติงาน หากทางผู้พัฒนาระบบมีการเปลี่ยนแปลงหน้าเว็บ ฟังก์ชันการเลือกแถวหรือการตรวจจับปุ่มดาวน์โหลดอาจต้องได้รับการอัปเดตตามลำดับ
